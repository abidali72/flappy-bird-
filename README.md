# 🐦 Flippy Bird - Android Edition

A modern, high-performance clone of the classic Flappy Bird game, built from the ground up using **Kotlin** and **Jetpack Compose**.

## 🚀 Features
- **Modern UI**: Pure Jetpack Compose implementation for smooth animations and transitions.
- **Immersive Experience**: Edge-to-edge display with hidden system bars for dedicated gameplay.
- **Game Engine**: Custom-built game loop logic using Kotlin Coroutines.
- **Responsive Design**: Adapts perfectly to various screen sizes and orientations.

## 🛠️ Tech Stack
- **Language**: [Kotlin](https://kotlinlang.org/)
- **UI Framework**: [Jetpack Compose](https://developer.android.com/compose)
- **Architecture**: MVVM
- **Navigation**: Compose Navigation
- **Theming**: Material 3
